특징:

- 특정 값을 여러 해시 함수에 넣어 나온 결과값들을 테이블에 인덱싱
- 충돌 가능성이 있기 때문에 False Positive (없는데 있다고 함) 발생 가능
- 반대로 False Negative (있는데 없다고 함)는 불가능
- 즉 `아마도 있다(...) / 없다`를 판별할 수 있다

- 구현이 매우 간단
- 해시 테이블에 비해 적은 양의 메모리를 사용
  - 왜? 값 자체를 저장하지 않음...
  - 비트맵에 1을 넣음

- URL이 악의적인 사이트인지 아닌지 파악을 위해 구글 크롬이 사용
